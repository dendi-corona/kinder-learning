# 🌟 Quick Start Guide - Kindergarten Learning Quest

## What is This?

A fun, colorful learning app for kids ages 4-6 with:
- ABC Adventure (letters A-Z)
- 123 Counting (numbers 1-20)
- Shapes & Colors
- Mini Stories
- Drawing Studio
- Rewards & Badges system

## 🚀 Running the App

### Option 1: Development Mode (Recommended)

```bash
cd kinder-learning
npm run dev
```

Then open **http://localhost:3000** in your browser.

### Option 2: Use the Start Script

```bash
cd kinder-learning
./start.sh
```

### Option 3: Static Build

The app is already built! You can serve the static files:

```bash
cd kinder-learning
npx serve out
```

Or deploy the `out/` folder to any static hosting (Netlify, Vercel, GitHub Pages, etc.).

## 📱 Installing as PWA

On tablets or phones:
1. Open the app in Chrome/Safari
2. Tap the menu (⋮ or Share button)
3. Select "Add to Home Screen" or "Install App"
4. Launch from your home screen like a native app!

## 🎮 How Kids Use It

1. **Home Screen**: Shows all learning modules with big, colorful buttons
2. **Pick an Activity**: Tap any module to start learning
3. **Earn Stars**: Complete activities to earn stars ⭐
4. **Get Badges**: Unlock achievements like "ABC Master" and "Number Ninja" 🏆
5. **Bottom Navigation**: Easy access to all modules from any page

## 👨‍👩‍👧 Parent Features

Visit the **Parents** section (gray button) to:
- See learning progress
- View earned badges
- Set daily time limits
- Export progress data
- Reset progress if needed

## 💾 Saving Progress

All progress is automatically saved! Kids can:
- Close the app and come back later
- Continue where they left off
- Keep their stars and badges
- Save their drawings

## 🎨 Features by Module

### ABC Adventure 🅰️
- Tap letters to hear them
- Learn words that start with each letter
- Trace letters with your finger
- Earn stars for completing letters

### 123 Counting 🔢
- Count fun objects (apples, balloons, stars!)
- Practice simple addition
- Tap numbers to hear them
- Complete all 20 numbers

### Shapes & Colors 🔺🔴
- Learn 6 basic shapes
- Learn 7 colors
- Play matching games
- Find shapes in real-world objects

### Mini Stories 📖
- Read 5 classic stories
- Answer fun questions
- Listen to read-aloud
- Earn Super Reader badge

### Drawing Studio 🎨
- Free drawing with colors
- Save your artwork
- Download drawings
- Create an art gallery

## 🎯 Tips for Parents

- **Encourage daily practice**: Even 10-15 minutes helps!
- **Celebrate achievements**: Cheer when they earn stars and badges
- **Use audio features**: Tap the speaker icon to hear pronunciation
- **Review progress**: Check the Parent Dashboard weekly
- **Export data**: Save progress reports to track growth

## 🔧 Troubleshooting

**App won't start?**
```bash
npm install
npm run dev
```

**Progress not saving?**
- Make sure you're using the same browser
- Check that localStorage is enabled
- Try the Parent Dashboard → Export to backup

**Audio not working?**
- Check browser permissions for audio
- Some browsers require user interaction first
- Try Chrome or Safari for best support
